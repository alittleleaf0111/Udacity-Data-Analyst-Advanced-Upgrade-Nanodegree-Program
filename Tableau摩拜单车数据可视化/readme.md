
##  Tableau摩拜单车数据可视化

### 项目说明
*
    使用 tableau 折线图、柱状图等可视化工具探索了上海市 2016 年 8 月摩拜单车数据，展现了摩拜单车在上
海市的发展趋势及使用情况。发现摩拜单车业务在八月份迅速增长， 且整体上骑行订单主要集中在上午 8
点和下午 5 点到 8 点， 上午的骑行时长小于晚间。

### 使用工具
*
    Tableau Public
